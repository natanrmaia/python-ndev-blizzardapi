Install
#######

.. toctree::
   :caption: Table of Contents
   :maxdepth: 3

Before installing, download and install Python. Version 3.11 or later is required.

We recommend using a virtual environment to install Python packages in an isolated location.

Pip
*************
To install the package, run the following command in the terminal:

.. code:: bash

    pip install python-ndev-blizzardapi

Source code
*************
This method is recommended for developers who want to contribute to the project.
If you just want to use the package, we recommend using the `pip <https://pypi.org/project/python-ndev-blizzardapi/>`_ method.

To install the package, run the following command in the terminal:

.. code:: bash

    pip install git+https://github.com/natanrmaia/python-ndev-blizzardapi.git

or download the source code (check the latest release in `github releases <https://github.com/natanrmaia/python-ndev-blizzardapi/releases>`_)
and run the following command in the terminal from the source code directory:

.. code:: bash

    python setup.py install
